## 종류
| 이름           | 바이트 수 | 표현형        |
| ------------ | ----- | ---------- |
| [[#boolean]] | 1byte | true/false |
| [[#byte]]    | 1byte | number     |
| [[#char]]    | 2byte | ascii      |
| [[#short]]   | 2byte | number     |
| [[#int]]     | 4byte | number     |
| [[#long]]    | 8byte | number     |
| [[#float]]   | 4byte | 소수         |
| [[#double]]  | 8byte | 소수         |


## 속성
### boolean
```java
boolean b = true;
```

### byte

### char
- 2byte 를 사용하며 UTF-4 를 쓰던가 아님 가변 문자 크기를 가졌던가 할텐데
- 기본적으로 [[ascii]] 문자를 사용
- [[ascii]] 문자는 기본적으로 1byte 만을 사용하기 때문에 요기에 추가로 뭔가 더 쓸 수 있는 언어가 있는데 이게 뭔지 모르겠네

### short
- 예전에는 기본 2byte 를 쓰는게 기본이었지만 이제는 컴퓨터 사양이 높아짐에 따라서 CPU 가 기본 4byte 크기를 사용하여 거의 사용하지 않는 데이터형

### int
- 4바이트 데이터 형
- 숫자를 표현하며 가장 기본적으로 많이 사용되는 데이터형

### long
- 8바이트 데이터형으로 사실 사용빈도가 높지는 않지만 ID 값을 기록하는 용도로 많이 사용됨

### float
- 4바이트 소수 표현 데이터형
- 소수 자체가 쓰는 상황이 많지 않다

### double
- 이제 슬슬 쓰기 귀찮다
- 8바이트 데이터형
- 소수를 표현하며 float 만큼이나 사용하지 않는달까
