# [[추상화 (Abstraction)]] 

> [!abstract] 
> 추상화 : 사물이나 표상을 어떤 성질, 공통성, 본질에 착안하여 그것을 추출하여 파악하는 것

추상화[^1]


공통성과 본질 특질을 모아서 추출 하는 것이 포인트


```java
interface Animal {
	void run(); // 달리기
	void howl(); // 울기
	void play(); // 놀다
	void eat(); // 먹다
}
```

이런 특질을 나타냄
