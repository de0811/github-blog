# Git Subtree
- 외부 저장소의 코드와 히스토리를 내 프로젝트에 **복사/병합(Copy/Merge)**하는 방식
## 🚨 사용 이점
- 외부 코드를 가져와 **내 프로젝트의 일부처럼** 통합하고 싶을 때
- 팀원들의 편의성이 중요하고 사용법이 간단해야 할 때
### 특징
- 포함된 후에는 일반 하위 폴더처럼 관리됨
- 별도 명령어 없이 `clone`만 하면 모든 코드 포함됨
- 외부 저장소의 커밋 히스토리까지 부모 프로젝트에 통합됨

### 주요 명령어
- **추가**: `git subtree add --prefix=<경로> <저장소 URL> <브랜치>`
- **업데이트**: `git subtree pull --prefix=<경로> <저장소 URL> <브랜치>`
