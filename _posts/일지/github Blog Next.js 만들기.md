# github Blog Next.js
- github Blog 를 Next.js 로 만들기
- 왜 Next.js로 하냐면 
	- 회사 작업에 필요할거 같아서 
	- React 없어졌대매?
	- 재미로?
	- 요즘 대세라고 해서?
## 이걸 왜 하게 되었냐
- 옵시디언에서 사용 중인 기능들이 되었으면 좋겠어
- 공부가 목적이지 블로그 쓰는걸 목적으로 하고 싶진 않아
	- 알아서 올려줘
- 나도 다른 블로그처럼 이쁘고 싶은데
- 댓글? 나도 되면 좋겠는데
- 이력서 대신 이걸 활용하는것도 좋을지도?
