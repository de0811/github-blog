# CodeStyler
코드 줄의 숫자 넣기
코드에 하이라이트 색상 넣기

```js hl:2,console error:9 title:"제목을 적는 구간" ref:[[2.Ref(데이터 및 정보 저장)/옵시디언/옵시디언|옵시디언]]
const a = 5
const b = 5
const c = 2


console.log(a + b)

function ac() {
 return a + c;
}

console.log(ac())
```

`{python}a = 10`
`{js}const a = 22`
`{python title:"제목 적는 방법"}print("테스트 제목")`
`{python icon title:"아이콘을 넣는 방법"}print("테스트 제목")`

`{java icon title:"비러머글 java" }log.info("뻐킹")`
