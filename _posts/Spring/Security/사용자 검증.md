# 사용자 검증

## AuthenticationManager
AuthenticationProvider 들을 관리하는 인터페이스
### ProviderManager
`AuthenticationManager`의 구현체로써 실질적인 소스
<span style="background:rgba(205, 244, 105, 0.55)">내용 보면 성공이라고 던져주는 모습이 보이는 것 보니 무조건 이걸 타긴 해야할듯 이건 커스텀 요소가 아닐 것으로 예상</span>
## AuthenticationProvider
```java
public interface AuthenticationProvider {
	Authentication authenticate(Authentication authentication) throws AuthenticationException;
	//return (UsernamePasswordAuthenticationToken.class.isAssignableFrom(authentication));
	boolean supports(Class<?> authentication);
}
```
### DaoAuthenticationProvider
`AbstractUserDetailsAuthenticationProvider` 의 구현체인 `DaoAuthenticationProvider` 에서 `userDetailsService` 통해서 검증을 진행
`additionalAuthenticationChecks` 함수에서 `PasswordEncoder` 를 이용해서 비밀번호 비교를 진행
