# 보안 사용자
### UserDetails 
```java title:"UserDetails"
public interface UserDetails extends Serializable {
	// 사용자에게 부여된 권한을 반환 (null 반환 불가)
	Collection<? extends GrantedAuthority> getAuthorities();
	
	String getPassword();

	String getUsername();

	// 계정이 만료 되었는지 여부 반환 (만료된 계정은 인증 불가)
	// 만료되지 않은 경우 true, 만료된 경우 false
	boolean isAccountNonExpired();

	// 계정이 잠겨있는지 여부 반환 (잠긴 계정은 인증 불가)
	// 잠기지 않았을 때 true, 잠겼을 때 false
	boolean isAccountNonLocked();

	// 사용자의 자격증명(비밀번호) 만료 되었는지 여부 반환 (만료된 자격 증명은 인증 불가)
	// 만료되지 않은 경우 true, 만료된 경우 false
	boolean isCredentialsNonExpired();

	// 사용자가 활성화되어 있는지 확인 (비활성화 계정은 인증 불가)
	// 활성화 되어 있다면 true, 비활성화시 false
	boolean isEnabled();

}
```
주석을 보면 `User` 클래스를 참조해서 만들어도 된다고 나와 있음
아니 주석이 영어라서 거의 볼 일이 없는데 망할
