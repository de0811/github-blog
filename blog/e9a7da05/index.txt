<h3 style="color: #C792EA;">
                  <span style="font-size: initial; color: #F0C292; margin-right: 0.5em;">h3</span>
                  char
                </h3>
<ul>
<li>2byte 를 사용하며 UTF-4 를 쓰던가 아님 가변 문자 크기를 가졌던가 할텐데</li>
<li>기본적으로 <a href="/github-blog/blog/3a7809ac">ascii</a> 문자를 사용</li>
<li><a href="/github-blog/blog/3a7809ac">ascii</a> 문자는 기본적으로 1byte 만을 사용하기 때문에 요기에 추가로 뭔가 더 쓸 수 있는 언어가 있는데 이게 뭔지 모르겠네</li>
</ul>
<h3 style="color: #C792EA;">
                  <span style="font-size: initial; color: #F0C292; margin-right: 0.5em;">h3</span>
                  short
                </h3>
<ul>
<li>예전에는 기본 2byte 를 쓰는게 기본이었지만 이제는 컴퓨터 사양이 높아짐에 따라서 CPU 가 기본 4byte 크기를 사용하여 거의 사용하지 않는 데이터형</li>
</ul>
<h3 style="color: #C792EA;">
                  <span style="font-size: initial; color: #F0C292; margin-right: 0.5em;">h3</span>
                  int
                </h3>
<ul>
<li>4바이트 데이터 형</li>
<li>숫자를 표현하며 가장 기본적으로 많이 사용되는 데이터형</li>
</ul>
<h3 style="color: #C792EA;">
                  <span style="font-size: initial; color: #F0C292; margin-right: 0.5em;">h3</span>
                  long
                </h3>
<ul>
<li>8바이트 데이터형으로 사실 사용빈도가 높지는 않지만 ID 값을 기록하는 용도로 많이 사용됨</li>
</ul>
<h3 style="color: #C792EA;">
                  <span style="font-size: initial; color: #F0C292; margin-right: 0.5em;">h3</span>
                  float
                </h3>
<ul>
<li>4바이트 소수 표현 데이터형</li>
<li>소수 자체가 쓰는 상황이 많지 않다</li>
</ul>
<h3 style="color: #C792EA;">
                  <span style="font-size: initial; color: #F0C292; margin-right: 0.5em;">h3</span>
                  double
                </h3>
<ul>
<li>이제 슬슬 쓰기 귀찮다</li>
<li>8바이트 데이터형</li>
<li>소수를 표현하며 float 만큼이나 사용하지 않는달까</li>
</ul>
